# 来源与可追溯性

## 课程标准

三科范围以教育部 2022 年义务教育课程方案和课程标准为主要依据：

- <https://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html>

仓库只保存课标定位 ID、短定位和原创拆解，不复制课程标准连续正文。

## 教材核验

- 数学使用人教版、北师大版和苏教版进行交叉核验。
- 语文使用统编教材版本进行位置和范围核验。
- 英语使用人教 PEP、外研版和译林版进行交叉核验。

教材的作用是验证知识是否真实出现、帮助查漏补缺和记录首次出现范围。
教材目录、单元和页码等事实元数据可以作为定位；教材内容本身不进入
公共仓库。`UNVERIFIED` 只表示当前没有取得足够证据，不表示知识不存在。

## 古诗词

古代作品本身可能已经进入公版，但现代整理、注释、排版和数字转录仍
可能具有独立权利。标记为 `CC-BY-SA-4.0` 的记录采用保守的共享方式，
保留相同许可边界；词义、理解、主题、情感和考点说明为项目原创整理。

## sourceRefs

每个实体和关系的 `sourceRefs` 保存上游审计中的稳定证据 ID，例如课标
锚点、公开来源或教材核验记录。它们用于追踪结论形成过程，不代表仓库
重新发布该来源，也不能单独替代对原始来源的检查。

社区修改来源结论时，应在 Pull Request 中同时提供可访问 URL、定位、
访问日期、证据能支持的范围和权利说明。维护者审核后才会合并到 `main`；
用户本地同步时还会再次由规则和模型核验。
